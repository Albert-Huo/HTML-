## 使用约束

- 生成初中物理实验 HTML 前，先按实验 `id`、`title` 或 `topic` 查询 `references/textbook-experiment-index.json`，并把匹配条目的器材、公式、步骤、观察/结论写入实验规格。
- JSON 匹配后必须再和 `references/初中物理实验目录-2023版摘录.md` 的目录实验名称/编号核对；如果索引缺失、匹配错位、目录实验与 JSON 条目不一致，必须回查相应教材 PDF 后再确定器材和步骤。
- PDF 回查优先级：
  1. 已知年级/册别时，优先查对应本地教材 PDF。
  2. 八上、八下 PDF 可文本提取时，优先使用文本搜索和页码证据。
  3. 九年级 PDF 为图片版时，使用页面视觉检查或 OCR；仍无法确认时，保留 `需核对` 标记。
- 回查教材 PDF 后，必须在实验规格和交付说明中记录 PDF 路径、页码或定位方式、找到的证据，以及它是覆盖还是补充 JSON 约束。
- 若 JSON 条目的 `raw_text` 与结构化字段不一致，以 `raw_text` 为准；若 `raw_text` 为空或条目标注 `需核对`，必须在交付说明中标注不确定性。
- 若教材页给出了明确器材与步骤，不得擅自增加非必要器材（如额外量筒、溢流流程）。
- 若需扩展为“可视化增强版本”，必须先交付一个“教材忠实版”，并在说明中标注扩展项。
