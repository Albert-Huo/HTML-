# 物理实验互动规则

## 目的

记录初中物理互动实验页面中可复用的交互规则，保证闯关模式、演示模式和自由模式的操作位置、反馈和验证逻辑一致。

## 上下文

适用于 `physics-middle/初中物理实验*.html` 单文件实验页面，尤其是包含 canvas 实验台、自动演示鼠标光标、实验面板和数据面板的页面。

## 规则

### 自动演示光标

- 演示模式中的鼠标光标必须移动到真实可命中的器材坐标，例如开关、电池组、线圈、小磁针或滑片中心。
- 不要用页面宽度比例估算点击位置来代替器材几何坐标；比例点容易与实际仪器错位。
- 自动演示触发动作时，应复用闯关模式的同一套操作函数或状态机，避免光标显示一个位置、状态却直接跳转到另一个场景。
- 每个关卡的演示目标坐标应从当前实验的几何函数计算，保证桌面端、移动端和 resize 后仍能对齐。
- 如果光标位置通过 `transform: translate(...)` 设置，点击动画不能再改写 `transform`，否则点击瞬间会覆盖定位并造成光标跳位；点击反馈应使用透明度、光晕或阴影变化。

## 示例

- 通电导线磁场实验中，演示光标依次指向小磁针、开关、电池组。
- 电动机换向器演示中，演示光标依次指向线圈、开关、电池组，并调用与用户点击相同的引导操作逻辑。

## 相关文档

- 当前规则与各实验页面内的 `demoGesture`、`hit*Part`、`operate*Part` 或引导操作函数相关。
